*** Gallery Shortcodes ***

Image Attachment Slideshow
[img_slides]

Options
None (Settings can be changed in Gallery settings per post).

Details
Creates an image slideshow of all attached images using Flexslider.  You can change the order in the Gallery settings per post.

*

Image Attachment Gallery
[img_gallery]

Options
None (Settings can be changed in Gallery settings per post).

Details
Creates an image gallery of all attached images.  You can change the order in the Gallery settings per post.

*** Video Shortcodes ***

Youtube Embed Shortcode
[youtube video_code="" width="" height=""]

Options
video_code="########" - the video ID that shows after watch?v=
width="560"
height="349"

Details
Embeds a Youtube video player using the ID specified in video_code option, in the specified width and height.
